🎓 KEY INSIGHTS

What Made This Work:
  1. Right model for each task (GPT-4 for article, Haiku for social)
  2. Batch API calls (5 social posts in 1 request)
  3. Template images instead of API (saves $2/episode)
  4. Extracted value (quotes from social batch)
  5. Clear cost tracking (per component)

What Was Avoided:
  ❌ Using GPT-4 for social posts (would add $1.50)
  ❌ Separate API calls per social platform (would add 5x cost)
  ❌ Replicate/Midjourney images (would add $2)
  ❌ Complex prompts (increases tokens)
  ❌ Unnecessary features (cost without benefit)
